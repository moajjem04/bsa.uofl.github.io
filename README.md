# bsauofl.github.io

A static GitHub Pages website for the Bangladeshi Student Association at the University of Louisville.

## What is included

- `index.html` — the main website page.
- `styles.css` — responsive styling for the directory.
- `script.js` — rendering and search behavior for profiles.
- `data/profiles.js` — editable current student and alumni profile data.
- `assets/` — profile picture placeholders that can be replaced with real member photos.

## Updating profiles

Edit `data/profiles.js` to add or update people. Each profile includes:

- `name`
- `status` (`current` or `alumni`)
- `title`
- `picture`
- `email`
- `linkedin`
- `tags`
- `bio`

## Publishing on GitHub Pages

For a user or organization site, rename the GitHub repository to `bsauofl.github.io` and publish from the default branch. GitHub Pages will serve `index.html` automatically.
